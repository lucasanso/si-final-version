Para utilizar os crawlers, primeiro crie um arquivo config.yaml seguindo a estrutura do config_example.yaml e insira as credenciais. Após isso, crie um ambiente virtual .venv e instale as dependências de requirements.txt. Após isso, instale o playwright com o comando playwright install no terminal. Após isso, utilize o comando scrapy crawl <nome do bot>. Lista de bots: bdf (Brasil de Fato), g1 (G1), folha (Folha de sao paulo), estadao (Estadão), monde (Le monde brasil diplomatique), correio (Correio do povo), carta (carta capital)
